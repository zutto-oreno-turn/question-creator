# question-creator
- create json from twitter

### Example
- https://www.zutto-oreno-turn.com/cdn/eigo/question/category/tweet/CNN.json

### Setup
```
$ git clone https://github.com/zutto-oreno-turn/question-creator.git
$ cd question-creator
$ npm install
```

### Run
```
$ cd question-creator
$ node app.js realDonaldTrump
$ node app.js CNN
$ node app.js unity3d
```
